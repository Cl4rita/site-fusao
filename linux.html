Sistema Operacional

O que é?

Um sistema operacional é um software ou conjunto de softwares que desempenha um papel fundamental na administração e gerenciamento dos recursos de um sistema. Esses recursos incluem componentes de hardware, sistemas de arquivos e programas de terceiros. O sistema operacional atua como uma interface entre o computador (ou dispositivo) e o usuário.

  • Assoc: Exibe ou modifica associações de extensões de arquivo com tipos de arquivo específicos
  • Cipher: Permite criptografar ou descriptografar arquivos e pastas no sistema de arquivos NTFS
  • Driverquery: Mostra informações sobre os drivers instalados no sistema
  • Comparação de arquivos (File Compare): Compara o conteúdo de dois arquivos e exibe as diferenças
  • Ipconfig: Exibe informações sobre configurações de rede, como endereço IP, máscara de sub-rede e gateway padrão
  • Netstat: Mostra estatísticas de rede, como conexões ativas, portas em uso e protocolos
  • Ping: Testa a conectividade com um host remoto, enviando pacotes ICMP e exibindo o tempo de resposta
  • Tracert (Trace Route): Rastreia a rota de pacotes até um host remoto, mostrando os saltos intermediários
  • Tasklist: Lista os processos em execução no sistema, incluindo seus IDs e uso de memória
