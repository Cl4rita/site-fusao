Cli e Gui

O que é?

Interface de Linha de Comando (CLI - Command-Line Interface) e Interface Gráfica do Usuário (GUI - Graphical User Interface) são duas formas distintas de interagir com um sistema operacional.

Cli

  • As CLIs são amplamente utilizadas por profissionais de tecnologia, incluindo desenvolvedores, administradores de sistemas, engenheiros de rede e especialistas em segurança cibernética.
  • Diferentes programas e bibliotecas oferecem suas próprias CLIs com comandos específicos, como ferramentas de versionamento de código, serviços em nuvem e gerenciadores de versões.
  • Os usuários interagem com o computador através de comandos de texto.
  • A representação visual ocorre em telas pretas com texto, frequentemente chamadas de terminais.

    • Exemplos: Terminal no macOS e Linux, e o Prompt de Comando no Windows.



Gui

  • É comum em sistemas operacionais de desktop, onde os usuários podem clicar em ícones para abrir aplicativos, arrastar e soltar arquivos e ajustar configurações através de menus.
  • É adequada para tarefas que exigem interação visual e não dependem de precisão de entrada manual.
  • É mais amigável para usuários que não têm conhecimento técnico profundo, pois não requer digitação de comandos.
  • A representação visual ocorreatravés de elementos visuais, como botões, ícones e janelas.

    • Exemplos: Área de Trabalho do Windows, a interface Aqua do macOS e os ambientes de desktop como GNOME e KDE no Linux.
