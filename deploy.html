Deploy

O que é?

Deploy é um termo utilizado no desenvolvimento de software para se referir à implantação de uma aplicação. Quando um programador conclui o desenvolvimento de uma solução, o deploy é a etapa em que essa solução sai do ambiente de testes e é disponibilizada no ambiente de produção, onde será efetivamente utilizada pelos clientes.

Consiste no processo de colocar no ar uma aplicação já concluída. Ele pode ocorrer durante várias fases do projeto, bem como após a sua conclusão.

Além disso, pode haver modificações no código e outras atualizações necessárias, como uma função no aplicativo ou website que ainda não existia. Para essas situações, é preciso fazer o deploy.
